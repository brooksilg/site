# brooks-ilg-site

My personal site written in Vue.js and built using Vue CLI.

[brooksilg.com](https://brooksilg.com/)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
